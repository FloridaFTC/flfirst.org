# Newsletter Summary Generation Prompt

You are tasked with creating a concise, well-structured summary of a Florida FTC newsletter. Your summary should follow this exact format and style:

## Format Requirements

1. **Header**: Use `## Newsletter #[Number] Summary` as the title
2. **Structure**: Organize content into clear sections using bold headers with colons
3. **Content**: Use bullet points for all information, maintaining a clean hierarchy
4. **Length**: Keep each section concise but comprehensive

## Content Guidelines

**Extract and organize the following key information:**

- **Deadlines and Important Dates**: Include submission deadlines, event dates, registration windows
- **Event Information**: Tournament dates, locations, registration fees, arrival times
- **Rule Changes and Updates**: Any new rules, requirements, or procedural changes
- **Competition Details**: Format changes, scoring updates, advancement criteria
- **Action Items**: Specific tasks teams/coaches need to complete
- **Key Numbers**: Team counts, advancement slots, registration fees, etc.

**Style Requirements:**

- Use clear, direct language
- Prioritize actionable information
- Include specific dates, times, and locations
- Maintain consistent formatting with colons after section headers
- Use bullet points for all details
- Keep paragraphs short and scannable

**Example Structure:**

```markdown
## Newsletter #[Number] Summary

**Section Name**:

- Key point 1
- Key point 2
- Key point 3

**Another Section**:

- Important detail 1
- Important detail 2
```

**Focus on:**

- Deadlines and time-sensitive information
- Competition logistics and requirements
- Rule changes and updates
- Specific action items for teams
- Important dates and locations

**Avoid:**

- Personal anecdotes or casual language
- Repetitive information
- Marketing language or promotional content
- Unnecessary context or background information

Create a summary that allows teams to quickly understand what they need to know and do, with all critical information easily scannable and actionable.
